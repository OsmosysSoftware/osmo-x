---
name: no-attribution
enabled: true
event: bash
conditions:
  - field: command
    operator: regex_match
    pattern: git\s+commit
  - field: command
    operator: regex_match
    pattern: (?i)(co-authored-by|signed-off-by|generated\s+(by|with)\s+(claude|ai|anthropic)|authored\s+by|created\s+by\s+(claude|ai)|noreply@anthropic|🤖\s+generated)
action: block
---

**BLOCKED: Attribution/author info in commit message**

Commit messages must NOT contain any author or attribution information such as:
- `Co-Authored-By: ...`
- `Signed-off-by: ...`
- `Generated by/with Claude/AI/Anthropic`
- `Created by Claude` or `Created by AI`
- `noreply@anthropic.com`

Remove all attribution lines and try again. Keep commits clean with just the conventional commit message.
