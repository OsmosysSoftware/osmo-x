---
name: no-attribution
enabled: true
event: bash
conditions:
  - field: command
    operator: regex_match
    pattern: git\s+commit
  - field: command
    operator: regex_match
    pattern: (?i)(co-authored-by|signed-off-by|generated\s+(by|with)|authored\s+by|created\s+by\s+claude|created\s+by\s+ai|noreply@anthropic|claude\s+code)
action: block
---

**BLOCKED: Attribution/author info in commit message**

Commit messages must NOT contain any author or attribution information such as:
- `Co-Authored-By: ...`
- `Signed-off-by: ...`
- `Generated by ...` / `Generated with ...`
- `Created by Claude` or similar AI attribution
- `noreply@anthropic.com`

Remove all attribution lines and try again. Keep commits clean with just the conventional commit message.
